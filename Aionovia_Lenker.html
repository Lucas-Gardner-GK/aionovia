🔄 GitHub flyt — push & pull

Slik syncer du endringer med Vetle

1
Du redigerer en fil via Claude
Be Claude om å endre noe i en fil som vanlig.
2
Push til GitHub
Åpne Terminal og kjør:

cd ~/Desktop/AIONOVIA_snarvei && git add . && git commit -m "oppdatering" && git push
3
Vetle henter endringene
Vetle kjører på sin Mac:

cd ~/Desktop/AIONOVIA && git pull
4
Vetle redigerer og pusher
Vetle ber Claude redigere, pusher med samme kommando som steg 2.
5
Du puller Vetles endringer
Du kjører:

cd ~/Desktop/AIONOVIA_snarvei && git pull
⚠️ Viktig regel: Ikke rediger samme fil samtidig. Si til hverandre hva dere jobber med — "jeg tar prototype", "jeg tar notater" — så unngår dere konflikter.
